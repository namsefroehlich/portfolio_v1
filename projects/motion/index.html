About Motion

Independently performed full-stack development of a social media app that enabled users to follow, create friend requests, and create, view, and like user posts. Built with React and Redux, Django DRF, PostgreSQL, with Docker containers on droplet servers with docker-compose managed by Gitlab runners.

React, Django, PostgreSQL, Docker

Challenge

Using React JS and Sass in Frontend, Django Rest Framework in the backend was intuitive until the Frontend was needed to be connected with the backend and Deployed with Docker and Git-Lab Runners for the first time.

Solution

After a lot of sweat & tears, I made it work. Learned the majority of the docker commands by heart, which helped me with the projects to come.
