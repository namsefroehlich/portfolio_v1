About Motion

Independently performed full-stack development of a social media app that enabled users to follow, friend request, and create, view, and like user posts. Built with React and Redux, Django DRF, PostgreSQL, with Docker containers on droplet servers with docker-compose managed by Gitlab runners.

React, Django, Postgres, Docker
